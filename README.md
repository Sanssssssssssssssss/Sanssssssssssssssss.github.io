# Personal Website

A simple static personal website that is ready for GitHub Pages.

## Files

- `index.html`: page structure and content
- `styles.css`: layout and visual styling
- `script.js`: small client-side enhancement

## Publish to GitHub Pages

1. Push this repository to GitHub.
2. Open repository `Settings`.
3. Go to `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/ (root)`.
6. Save and wait for the site URL to be generated.

## Customize

- Replace `Your Name` with your real name.
- Replace `YN` with your initials or swap the visual block for a real photo.
- Update the email and social links in the contact section.
- Rewrite project cards with your real work.
